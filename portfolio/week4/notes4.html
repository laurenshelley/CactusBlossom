Object-oriented Programming
-The main concepts are encapsulation, polymorphism, and inheritance.
Encapsulation
-This is the method of building data and methods that work with that
data into a single unit. Polymorphism
-This method explains that different objects can be used and even
share the same method. Inheritance
-This is the process of taking bits and pieces of one method and
incorporating it into another. This way you can recycle and improve old methods without having to start completely from scratch. Classes in javascript
-Classes are used to name different objects. By naming the objects you
are able to call the objects in javascript as well as CSS. Chaining Functions
-This is where one object can call multiple functions at a time helping
organize the code.
